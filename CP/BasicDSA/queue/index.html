    Queue

    Queue是一種資料結構，像是隊列一樣，先進的在隊列的頭，人要出來的話當然是先進的人先出。

    Queue in C++

    C++的STL就有了，直接拿來用吧

    一樣，這裡T指的是塞進去資料的Type

    • 宣告: queue<T> q;

    • 放在queue的最後面: q.push(...);

    • 拔掉queue第一個元素: q.pop(...);

    • 存取第一個的元素: q.front();

    • 存取最後一個的元素: q.back();

    • 輸出: for(auto t:q) cout << t;

    • 其他: empty,size之類的要用再查

    • Reference Document: https://en.cppreference.com/w/cpp/container/queue

    Queue的應用

    1. 當你需要這種隊列的結構的時候。(廢話)

    2. BFS: 啊啊啊這也是long topic，不在此"基礎"的計畫內(雖然是真的蠻基礎的)，但是計畫的時間不夠了，會之後再補上相關內容。

    3. SPFA(Bellman Ford with queue optimization): 其實SPFA根本也是BFS的變形的一種吧。

    廣義Queue: Deque

    因為Queue這樣感覺操作有點少，有沒有頭尾都能修改的結構呢?

    有的，就是Deque!

    操作就是頭尾都能修改，雙向的了

    deque<T> dq;,dq.push_front;, dq.pop_front;, dq.push_back;, dq.pop_back;之類的，用法差不多。

    https://en.cppreference.com/w/cpp/container/deque

    所以各位有沒有發現stack跟queue其實就是deque砍掉幾個功能而已。

    Deque題目

    我在這邊放兩個簡單的Deque題

    Permutations

    注意\(n=1\)有解(小心這個case XD)，\(1<n<4\)顯然列舉發現沒有解，\(n=4\rightarrow 2,4,1,3\)，\(n>4\)的話開一個std::deque先把\(2,4,1,3\)入隊，接下來每個數字依序放到隊伍之末，插進隊伍之頭...交替地作，這樣讓相鄰的兩個數字強制分開，最後輸出std::deque就好啦

    Palindrome Reorder

    小Case先特判掉，然後考慮一般的Case。我們先統計每個字母出現了多少遍，注意由對稱性我們知道總出現次數為奇數的字母只能有一個。然而要怎麼輸出我們想要的順序(回文)呢?有請我們的老朋友出場-Deque! 先把std::deque塞奇數個的那字母(因為他在中間)，然後之後把其他的字母一直以push_back跟push_front交替插入deque的頭跟尾就好啦 然後這題實作偏煩，要統計蠻多東西的，Debug蠻久的。
