    Vector - 進擊的陣列

    就是進可攻(可變大小)退可守(也可以固定大小)的array，我推薦它。

    直接放用法了啦

    宣告

    若是固定大小為n的int vector:

    vector<int> arr(n,0);

    後面的0代表初始化為0。

    後方加上元素

    arr.push_back(...);
//or
arr.emplace_back(...);

    雖然兩者背後的運行細節有差，但速度沒什麼差，都好。

    不過push_back(或是emplace_back)因為牽涉到記憶體重新分配(因為不知道整體多長所以當長度快到的時候會多開記憶體)，所以不要push_back 太多次，不然時間很感人。

    其他

    • arr.resize(k,0); 延伸長度為k，被創建新的東西設為0

    • 可用reserve保留空間。

    • arr.assign(...); 之類的，要用再查吧

    • arr.begin(),arr.end()之類的iterator。(我記得end是指向最後一項的下一位，要注意)

    • 直接看Document啦 https://en.cppreference.com/w/cpp/container/vector

    二維陣列

    有點長，但打久了自然就熟能生巧了

    vector<vector<int>> arr(n,vector<int>(n));

    Ordinary

    大部分的教程輸出都是這樣教

    for(vector<int>::iterator it=v.begin();it!=v.end();++it){
        cout << *it << '\n';
}

    りしれ供さ小

    這根本就是在勸退人吧。

    其實它就是枚舉指針然後去輸出指針的值，因為一般我們是枚舉每個位子的值。

    在古老的C中確實調用指針比較value快，但是現在是2022欸，根本沒差，而且有夠反人性(懶惰)。

    所以它其實可以跟一般陣列一樣cin >> arr[i]; cout << arr[...];之類的，讚。

    現在是2022欸

    或者是你如果真的非常懶惰，而且你的C++是現代的C++可以用

    vector<int> arr(n);
for(auto &t:arr) cin >> t; //注意那個&
for(auto t:arr) cout << t;

    這是簡潔，不是懶惰!
