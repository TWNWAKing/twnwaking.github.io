    分治 Divide and Conquer

    字面上的意思來看，就是Divide和Conquer這兩個動作。

    Divide就是把大問題切成小問題(通常是對半切)，注意這邊的小問題其實沒有什麼限制(?)，只要確保是可以劃分成子問題的，且之後能合拼出正確答案就好了。

    Conquer就是瘋狂解決小問題，然後設法合併起來，Done。

    我來展示一點比較"程式"的Demonstration

    1. 設我們的問題範圍左界是\(l\)右界\(r\)，\(m=\lfloor\dfrac{l+r}{2}\rfloor\)

    2. 把\([l,r]\)切成\([l,m],[m+1,r]\)解它們

    3. 合併Merge \([l,m],[m+1,r]\)

    然後第二步會一直劃分下去直到變成\(l=r\)之類的小問題，就好求解了

    分治的應用

    • Merge Sort

    • 線段樹，好毒喔(分治的一個經典實體化產物，至於它的介紹之後出現在這個Blog的某個地方)

    • 二分搜: 基本上二分搜也算是一種分治(?)，因為它每次Merge的時候會把Target不在的那一半丟掉。

    • 快速冪

    inline long long mod_qpow(long long a,long long b,long long m){ //Mod quick power
    a%=m;
    long long tmp=1;
    while(b){
        if (b&1) tmp=tmp*a%m;
        a=a*a%m;
        b>>=1;
    }
    return tmp;
}
