    副標題(這裡可以打幹話)

    競賽規則

    請務必詳讀競賽規則以保障考生之權益。

    1. 競賽時間為日期，共幾小時。

    2. 本競賽採用電腦線上自動評分機制，程式必須依規定上傳至評分主機評分。請考生針對每個題目在規定時間內按題目要求撰寫正確的程式碼，並交由線上評分系統後，系統將將自動編譯並進行測試，裁決您所提交的程式是否為能正確地解決問題以及不超過題目限制之時間和空間限制之程式，並且根據通過的子任務數打分。

    3. 本次競賽試題共幾題，每題皆有子任務。

    4. 為了愛護地球，本次競賽題本僅提供電子檔，不提供紙本。

    5. 每題的分數為該題所有子任務得分數加總；單筆子任務得分數為各筆繳交在該筆得到的最大分數；每題最終分數以該題所有單次評分結果之最高分計算。

    6. 程式可重複上傳及評分，但每一題在每次提交後都有的冷卻時間幾秒，期間暫時不能提交您的程式碼，裁判得視情況調整冷卻時長。

    7. 全部題目的輸入/輸出皆為標準(Standard)輸入/輸出。

    8. 所有輸入輸出請嚴格遵守題目要求，多或少的換行及空格皆有可能造成評分系統判斷為答案錯誤，該題將可以0分計算。

    9. 競賽系統使用與全國學科能力競賽/資訊奧林匹亞相同的CMS線上評分系統，評測伺服器為Ubuntu幾，編譯器版本為g++ 5.4.0 / gcc 5.4.0，編譯參數請見評分系統的題目敘述頁。

    10. 所有試題相關問題請於競賽系統中提問，題目相關公告也會公告於競賽系統，請密切注意。

    11. 如有電腦問題，請舉手向監考人員反映。

    12. 如有如廁需求，須經過監考人員同意方可離場。

    13. 考生除必要文具外，不得攜帶任何其他物件，如隨身碟、行動電話、計算機、英文字典、手冊、電子通訊器材或電腦週邊等。

    14. 不得攜帶任何參考資料，但競賽系統上或是由主辦單位核可提供的參考資料可自行參閱，參考檔案位於評分系統的參考(Reference)頁或是由主辦方統一公告。

    15. 不得使用任何非主辦方核可的來自外部已經預先撰寫好之程式碼。

    16. 競賽中請勿交談。請勿做出任何會干擾競賽的行為。

    17. 如需使用 C++ 的std::cin或std::cout可將以下程式碼插入main函數以及將endl取代為'\n'來優化輸入輸出速度。唯須注意不可與scanf/printf混用。

    std::ios::sync_with_stdio(0);
std::cin.tie(0);
    1. 如需使用C++的scanf或printf需在程式碼開頭引入標頭檔#include <cstdio>。scanf與printf的使用方式在下方有做簡略的陳述。

    scanf常用的讀入方式如下：

    scanf("%d",&x); //讀入一個有號整數至int型態變數x。
scanf("%lld",&x); //讀入一個有號整數至long long型態變數x。
scanf("%u",&x); //讀入一個無號整數至unsigned int型態變數x。
scanf("%llu",&x); //讀入一個無號整數至unsigned long long型態變數x。

    printf常用的輸出方式如下：

    printf("%d\n",y); //輸出一行包含一個int型態變數y。
printf("%lld\n",y); //輸出一行包含一個long long型態變數y。
printf("%.5f\n",y); //輸出一行包含一個double型態變數y到小數點後5位（可自行將5換成其他正整數）。
printf("%.5Lf\n",y); //輸出一行包含一個long double型態變數y到小數點後5位（可自行將5換成其他正整數）。
printf("%.5e\n",y); //以科學記號表示法輸出一行包含一個double型態變數y ，尾數輸出到小數點後5位（可自行將5換成其他正整數）。
printf("%.5Le\n",y); //以科學記號表示法輸出一行包含一個long double型態變數y，尾數輸出到小數點後5位（可自行將5換成其他正整數）。

    1. 請勿且嚴禁以各種惡意方式（包括但不限於在短時間內刻意大量重整頁面）攻擊評測主機。若發現有相關行為則先會給予口頭警告，若屢勸不聽則將以《刑法》第360條：「無故以電腦程式或其他電磁方式干擾他人電腦或其相關設備，致生損害於公眾或他人者，處三年以下有期徒刑、拘役或科或併科十萬元以下罰金。」做出告訴處理。

    2. 對於整數的輸入，如果您認為您的代碼時間複雜度正確但是仍然超時，可以嘗試使用整數的快速讀入： 

      inline int read(){
    int x = 0, Sign = 1;
    char c = getchar();
    while(c < '0' || c > '9'){
        if(c == '-') Sign = -1;
        c = getchar();
    }
    while(c >= '0' && c <= '9'){
        x = x * 10 + c - 48;
        c = getchar();
    }
    return x * Sign;
}
      函數返回值為讀入的第一個整數。請注意，快速讀入作用僅為加快讀入，並非強制使用，考生須全權承擔使用此程式碼之後產生之所有後果的責任，主辦單位亦不會對使用此讀入函數後產生任何錯誤的後果而負責。

    3. (特殊規則)為了鼓勵考生使用資料結構解題，主辦方將提供AtCoder(AC) Library資料結構模板庫來降低使用和實作資料結構的門檻，使考生能專注在邏輯的思考上。使用方法以及模板庫可用之功能已經在主辦方提供之檔案內有說明文件可做參考。請注意，此模板庫為主辦單位核可之預先邊寫好的程式碼，可在賽中使用，但並非強制使用，考生須全權承擔使用之後產生之所有後果的責任，主辦單位亦不會對使用此模板庫後產生任何錯誤的後果而負責。
